4 `ns3-bgp` is a BGP speaker application for `ns-3`. It is built on top of [libbgp](https://github.com/Nat-Lab/libbgp). For simple usage and quick start, refer to examples. For detailed API usages, refer to document.
5 
6 ### Install
7 
8 To install `ns3-bgp`, first install [libbgp](https://github.com/Nat-Lab/libbgp/) 0.6.x. Then navigate to your ns-3 source directory and clone `ns3-bgp` as `bgp` in `src/` and configure `ns3`:
9 
10 ```
11 $ git clone https://github.com/Nat-Lab/ns3-bgp src/bgp
12 $ ./waf configure
13 ```

15 ### Log Components
16 
17 The following log components are avaliable.
18 
19 - `Bgp`: The BGP speaker application.
20 - `BgpLog`: The log forwarder to forward `libbgp` log to `ns-3`
21 - `BgpRouting`: The BGP routing protocol (`ns3::Ipv4RoutingProtocol`).
